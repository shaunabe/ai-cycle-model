# AI Cycle Model

Interactive 24-month scenario model for public and private AI asset pricing.
Single static file — no build, no dependencies, no keys.

**Model:** price = revenue surprise (actual growth vs ~40%/yr already priced in)
× sentiment multiple (mean-reverts to a justified level, overshoots in panic).
Private marks chase public with a lag and downside stickiness.

**Levers:** open-source share, token deflation, demand elasticity,
infrastructure constraint, enterprise productivity realization, startup-led
adoption share. Five presets; overlays for Nasdaq 2000, the 1998–2000 melt-up,
and the 2021 cloud drawdown.

Deterministic toy model for scenario discussion — not investment advice.
